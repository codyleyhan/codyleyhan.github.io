Some of my projects.

Visualizing LA

Description:

Developed a visualiziation of the parking tickets that occur in the city of Los Angeles using data from Los Angeles' open data API. The visualization attempts to visualize where people are most likely to get parking tickets in an interactive and visually appealing way. The interaction occurs by interacting with a map and interctive hexagonal heatmap. Created working with the Creative Labs Fall 2016 advanced project team.

Technologies:

  • Leaflet.js
  • D3.js
  • Mapbox.js
  • jQuery
  • HTML5
  • CSS3
  • Responsive Design
  • Agile Development

fixMoto API

Description:

Planned and developed a modern REST API back end for the fixMoto application. The API was designed with cross platform use as the main driving force. The API can be utilized by a web application or a iOS or Android application while keeping data concurrent and using the same account. This allows for better ingeration of the application across all devices.

Technologies:

  • Node.js
  • REST Architecture
  • JavaScript ES6
  • RethinkDB
  • JSON Web Tokens
  • Express.js
  • Mocha.js/Chai.js

Visualization of Genetic Algorithm

Description:

Simple path planning algorithm that navigates around a random obstacle. The algorithm utilizes a Genetic Algorithm with techniques of population, mutation, and crossover occurring that attempts to model the real world. The genetic algorithm is a form of artificial intelligence that is used to solve non linear problems. This projects was inspired by and developed using models by Daniel Shiffman.

Technologies:

  • Java
  • Processing
  • Genetic Algorithm
  • Artificial Intelligence
